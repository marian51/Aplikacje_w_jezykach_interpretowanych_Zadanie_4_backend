# zadanie-sklep

## Instalacja projektu
```
npm install
```

## Konieczne jest uruchomienie najpierw XAMPP i PRZED uruchomieniem forntendu
> **Aplikacja powinna działać na porcie 8081 (`localhost:8080/`)**

### Uruchomienie
```
node server.js
```
